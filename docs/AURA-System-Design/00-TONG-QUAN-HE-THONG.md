## 1. Tầm Nhìn Sản Phẩm

### Định Nghĩa
AURA (Ambient Understanding & Resonance Application) là nền tảng mạng xã hội thế hệ mới, sử dụng **trí tuệ cảm xúc xung quanh** (Ambient Emotional Intelligence) để **tự động hiểu** trạng thái cảm xúc người dùng thông qua phân tích hành vi sâu, từ đó dẫn dắt toàn bộ trải nghiệm: từ nội dung hiển thị, kết nối con người, đến bảo vệ sức khỏe tâm lý.

### Sự Khác Biệt Với AURA v2.0

| Khía cạnh | v2.0 (Cũ) | v3.0 (Mới) |
|---|---|---|
| **Cơ chế cảm xúc** | User tự chọn mood (8 options cứng nhắc) | AI tự suy luận từ hành vi đa tín hiệu |
| **Feed** | Sort by time + mood filter | Deep Recommendation Pipeline + Emotional Resonance |
| **Matching** | Rule-based mood compatibility | Soul Connect (deep emotional pattern matching) |
| **Communities** | 8 Vibe Rooms cố định | Emotional Waves (tự sinh từ AI, ephemeral) |
| **Wellbeing** | Không có | Wellbeing Guard + Emotional Mode Detection |
| **Identity** | Mood emoji/text | Aura Ring (visual gradient hào quang) |

### Tuyên Bố Sứ Mệnh
> *"AURA doesn't ask how you feel — it already knows. And it connects you with the right people, at the right moment, in the right way."*

### Tagline
> **"Your Emotions. Your Feed. Your People."**

## 5. Quyết Định Kiến Trúc Chiến Lược

### Tổng Kết Lựa Chọn Công Nghệ

AURA v3.0 sử dụng kiến trúc **Hybrid**: **Firebase** cho data layer + real-time + authentication, kết hợp **FastAPI (Python)** backend riêng cho toàn bộ AI pipeline. Cách chia này tối ưu hiệu năng AI (không bị cold start, không giới hạn memory/timeout) trong khi vẫn tận dụng Firebase cho real-time và mobile SDK.

| Thành phần | Lựa chọn | Lý do |
|---|---|---|
| **Frontend** | Flutter (Dart) | Đa nền tảng, tích hợp Firebase SDK, on-device AI |
| **State Management** | Riverpod | Linh hoạt, type-safe, phù hợp reactive data streams |
| **Authentication** | Firebase Auth | Google/Apple/Email sign-in, tích hợp sẵn |
| **Database chính** | Cloud Firestore | NoSQL realtime, offline sync, security rules |
| **Realtime Chat** | Firebase Realtime Database | Tối ưu cho high-frequency read/write |
| **File Storage** | Firebase Cloud Storage | CDN tự động, quy tắc bảo mật |
| **Push Notification** | Firebase Cloud Messaging (FCM) | Ổn định, miễn phí, iOS/Android |
| **AI Backend** | FastAPI (Python 3.12) | Dedicated server cho AI, không bị cold start, unlimited memory/timeout |
| **AI/ML – Emotion Inference** | FastAPI + HuggingFace | Sentiment analysis, emotion fusion, behavioral analysis |
| **AI/ML – Recommendation** | FastAPI + Custom scoring | Multi-stage recommendation pipeline |
| **AI/ML – Future** | FastAPI + Vertex AI / TFLite on-device | FER, Keystroke, GNN, Multimodal Fusion |
| **Firebase Triggers** | Cloud Functions (Python) | Lightweight triggers → gọi FastAPI, event sync |
| **Behavioral Tracking** | Firebase Analytics + Custom Events | Track implicit signals (scroll, dwell) |
| **Hosting** | Firebase Hosting + Cloud Run | Admin panel web + FastAPI deployment |
| **Feature Flags** | Firebase Remote Config | A/B testing, gradual rollout |
| **Monitoring** | Firebase Analytics + Crashlytics + Performance | Full observability |

## 9. Tại Sao AURA v3.0 Sẽ Thành Công

### Insight Cốt Lõi
> Các nền tảng hiện tại **khai thác cảm xúc** để tạo engagement (exploit).  
> AURA v3.0 **hiểu cảm xúc** để tạo **meaningful connection** (understand).  
> Đây là sự chuyển đổi từ **Attention Economy → Emotional Economy**.

### Lợi Thế Cạnh Tranh

1. **First-mover trong "understood emotions"**: Không nền tảng nào tự suy luận cảm xúc + dùng nó cho wellbeing
2. **Thuật toán có đạo đức**: Wellbeing Guard là bằng chứng nền tảng quan tâm sức khỏe tâm lý
3. **Viral by design**: Aura Ring tạo visual identity mạnh, dễ chia sẻ, dễ nhận diện
4. **Network effect**: Emotional Waves tạo FOMO lành mạnh ("20 người đang cùng cảm xúc với bạn!")
